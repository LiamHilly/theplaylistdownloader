HOW TO USE yt-dlp WITH COOKIES FOR PRIVATE SOUNDCLOUD PLAYLISTS
---------------------------------------------------------------
1. Place yt-dlp.exe in this folder (if not already).
2. Make sure 'soundcloud_cookies.txt' is in this folder.
3. Open CMD here (Shift + Right Click > "Open PowerShell window here")
4. Run the following command replacing <PLAYLIST_URL>:

yt-dlp.exe --cookies soundcloud_cookies.txt <PLAYLIST_URL>

---------------------------------------------------------------
This allows downloading of private playlists using your SoundCloud cookies.
